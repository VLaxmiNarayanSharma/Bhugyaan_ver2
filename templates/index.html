GEE Connect (Earth Engine + Drive)

Provide a service account JSON with Earth Engine + Drive permissions for exports.

Service Account Authorization Guide (Recommended)

  1. Create service account: Google Cloud Console → IAM & Admin → Service Accounts → Create Service Account. Use project ee-laxminarayan090503, name bhugyaan-api, assign role Owner (initial setup).
  2. Download JSON key: Open the service account → Keys → Add key → Create new key → JSON.
  3. Enable APIs: APIs & Services → Library → enable Earth Engine API and Cloud Storage API.
  4. Grant Earth Engine access: In Earth Engine Code Editor Assets settings, add service account email like bhugyaan-api@ee-laxminarayan090503.iam.gserviceaccount.com.
  5. Remove old local credentials (important): delete C:\Users\sharm\.config\earthengine\credentials to avoid conflicts.
  6. Use this UI: Enter Project ID, upload downloaded JSON, click Authorize & Connect.
  7. Restart server: uvicorn main:app --reload --port 1200.

After setup, test /lulc-timeseries, /download, and /generate-lulc.
